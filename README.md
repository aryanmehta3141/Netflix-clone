# Netflix Clone

A responsive Netflix-inspired landing page built using HTML and CSS. This project recreates the look and feel of Netflix's homepage while focusing on modern UI design, responsive layouts, and clean frontend development practices.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Modern Netflix-inspired user interface
* Multiple content sections
* Hover effects and smooth transitions
* Flexbox and CSS Grid based layouts
* Clean and organized code structure

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid

## Learning Outcomes

Through this project, I practiced:

* Building responsive web layouts
* Using Flexbox and Grid effectively
* Creating modern UI components
* Writing maintainable HTML and CSS
* Improving frontend design skills
